Практика підключення шрифтів
в HTML + CSS для домашнього завдання

Встановлення розміру шрифту

Маленький текст - font-size: 12px

Звичайний текст - font-size: 16px (за замовчуванням)

Великий текст - font-size: 24px

Встановлення стилю шрифту

font-style: normal - Звичайний текст

font-style: italic - Курсивний текст

font-style: oblique - Похилий текст

Встановлення насиченості (товщини) шрифту

font-weight: 100 — Thin

font-weight: 300 — Light

font-weight: 400 — Regular (normal)

font-weight: 500 — Medium

font-weight: 700 — Bold

font-weight: 900 — Black

Міжрядковий інтервал

line-height: 1 - Текст без відступів між рядками.

line-height: 2 - Великий відступ між рядками

Інтервал між літерами та словами

letter-spacing: normal — ЗВИЧАЙНИЙ ТЕКСТ

letter-spacing: 5px — Р О З Т Я Г Н У Т И Й

letter-spacing: -1px — Стиснутий текст

word-spacing: 10px — Встановлення інтервалу між словами

Оформлення тексту

text-decoration: underline - підкреслений текст

text-decoration: overline - лінія над текстом

text-decoration: line-through - закреслений текст

text-decoration: none - без оформлення

text-decoration-style: wavy - хвиляста лінія під текстом

text-decoration-style: dashed - пунктирна лінія під текстом

text-decoration-style: dotted - крапкова лінія під текстом

text-decoration-style: double - подвійна лінія під текстом

Встановлення типу (регістр) літер (прописні або з великої літери)

text-transform: uppercase - всі великі

text-transform: lowercase - всі малі

text-transform: capitalize - перша буква кожного слова

text-transform: none - без змін

Управління поведінкою пробілів (white-space)

white-space: normal - стандартна поведінка, пробіли зливаються, текст переноситься

white-space: nowrap - текст не переноситься на новий рядок.
